# 复杂反转创意方法论（Complex-Reversal Concept Pack）

> 来源：基于同事的 `complex-drama-creator-skill`（面向海外短剧的 5–7 层反转方法论）改写。
> 适用：`creative` 阶段的「复杂反转」可选模式 / `overseas` 海外加强 / 红果批量工厂的"高反转密度"批次。
> 不是默认风格，只在用户开启该模式时按需加载。

---

## 1. 这套方法论解决什么问题

普通爽剧靠 1–2 个强反转 + 高密度钩子就能跑通付费曲线；但当目标是 **海外平台（ReelShort / DramaBox / Dreame / TikTok）的复杂题材** 或 **想做"被反复颠覆"质感的高概念短剧**时，反转密度需要拉到 **5–7 层**，并且层与层之间要有逻辑递进。

这套方法论给的是反转密度的"配方表"：

- 反转**从哪些维度抽**（工具箱）
- 反转**怎么排列**（三幕递进）
- 角色**怎么写**才能视觉化（5 要素模板）
- 哪些是**反陷阱**（避免常见翻车）

## 2. 反转工具箱（7 类）

每一层反转都应来自下面任一类型，**不要同类反转连续叠**（例如连续两层"身份反转"会让观众疲劳）：

| 类型 | 描述 | 触发钩 |
| --- | --- | --- |
| **身份反转** | 好人其实是坏人 / 坏人其实是好人 / 配角才是主谋 | "他原来是……" |
| **关系反转** | 最亲近的人其实是敌人 / 陌生人原来是关键亲人 | "你才是真正的……" |
| **动机反转** | 表面动机是伪装，真实动机完全不同 | "我做这一切，是因为……" |
| **现实反转** | 整个故事其实是模拟 / 回忆 / 梦境 / 实验 | "这一切根本没发生" |
| **存在反转** | 主角不是人类 / 是 AI / 是时间循环单位 / 已死亡 | "你以为你是……？" |
| **世界观反转** | 故事赖以成立的核心规则被推翻 | "这个世界从来不是你想的那样" |
| **元叙事反转** | 整个故事是书中书 / 剧中剧 / 病历 / 监控录像 | "你现在看的，就是……" |

**密度建议**：

- 5 层为下限，7 层为上限。
- Act 1 至少 1 层（小反转，建立期待）。
- Act 2 至少 2 层，含一次"重大颠覆"（关系或身份级）。
- Act 3 至少 1 层"终极反转"，**但不要让终极反转推翻一切让观众觉得被骗**（最常见的翻车，下面反陷阱里再讲）。

## 3. 三幕反转链结构

每个 Act 必须写成 **完整段落**（250–300 字 / Act），不是要点列表。每段必须包含：具体场景、人物动作、关键对白、当前层反转的揭晓时刻。

### Act 1（铺垫）

- 表面故事和世界观成立 → 出现异常 / 疑问 → **第一层小反转**收尾
- 小反转应揭示"这件事比看起来复杂"，但**不要**马上动摇主线信任

### Act 2（深化）

- 主角调查 / 逃避 / 误判 → 敌人或环境升级 → **关系或身份的重大反转**
- 这一幕至少给到 2 层反转，且其中一层必须**重新定义观众对前面剧情的理解**（"原来 Act 1 的那场戏其实是……"）
- Act 2 结尾必须是全剧最密集的爆点，撑起付费节点

### Act 3（颠覆）

- 终极反转 → 颠覆故事前提 → 上升到主题 / 哲学 / 元叙事层面
- **必须闭环**：反派处置明确、核心谜团全部揭晓、不留第二季钩子
- **不要让观众觉得"前面都白看了"** —— 终极反转应该让观众想"重看一遍"，而不是"被骗了"

## 4. 第一主角视觉化模板（5 要素）

主角描述必须**让导演 / 选角 / 封面摄影一眼能想象出来**，固定写 5 项：

```
[名字]——[年龄]，[外貌核心特征]，[穿着风格]，[气质特点]，[整体印象一句话]
```

示例：

> Lucas——28 岁，瘦削敏感的记忆架构师，黑框眼镜下藏着忧郁的眼神，常年灰色高领毛衣，手指修长适合键盘操作，看起来既脆弱又坚韧。

> Mia——24 岁，体型纤细却眼神锐利的私家侦探，深褐色波浪短发，黑色机能风夹克 + 工装裤，走路带风但说话克制，第一眼像不爱说话的猫。

**禁止**：泛泛的"高冷霸总""美丽善良"，无法成像。

## 5. 英文剧名 + 海外人物本土化（仅适用于海外向）

### 5.1 剧名风格

适合海外平台的短剧名特征：

- **短促有力**：`GLASS PRISON` `THE THIRD EYE` `DOUBLE MURDER`
- **诗意化**：`THE DREAM THIEF` `THE MIRROR GAME`
- **悬疑式**：`THE CHOICE MAKER` `THE LIE DETECTOR`
- **复杂组合**：`THE MEMORY MARKET OF LOST SOULS`
- **矛盾修辞**：`THE TIME MERCHANT`

写法要求：

- 必须能**单独承担 ReelShort 封面投流**（一行 + 一张脸即可成立）
- 暗示主题或核心反转之一，但**不剧透终极反转**
- 全大写，4 词以内最稳，不得带任何中文字符

### 5.2 角色与世界本土化（硬性约束）

海外向必须**完全本土化**，**严禁任何亚裔元素**：

- 所有人名只用纯英文名（First + Last）：`Mia Carter` / `Lucas Reed` / `Ethan Walker` / `Sophia Hayes` / `Damian Cross`。
- 禁止：中文名、拼音名（`Lin Xia` / `Wang Wei`）、双名格式（`林夏 / Lin Xia`）、亚裔姓氏（Chen / Wang / Li / Park / Kim / Nguyen / Tran / Suzuki）。
- 角色外貌不得使用「黑长直 / 杏眼 / 东方面孔 / 旗袍 / 汉服 / 韩服」等亚裔/东方暗示。
- 城市、家庭结构、节庆、教育体系全部走海外语境（NYC / LA / Chicago / London / Sydney 等），禁止「春节 / 高考 / 体制内 / 相亲角 / 婆媳」等中式元素。
- 如果原始 brief 或上游素材里出现亚裔人物或中国语境，**必须替换**为西方角色与海外语境，不要"忠于原作"地保留。

## 6. 反陷阱清单（绝对避免）

- ❌ **反转为反转而反转**：层与层之间没有逻辑链接，纯靠"再来一次"
- ❌ **终极反转推翻一切**：让观众觉得"我前 50 分钟看了什么"
- ❌ **角色形象模糊**：写完三幕没人能想象主角长什么样
- ❌ **Act 写成要点**：分点 / 罗列，没有具体动作和对白
- ❌ **同类反转连击**：例如 Act 1 身份反转、Act 2 还是身份反转
- ❌ **强中国地域设定**：海外向必须本土化人名 / 城市 / 阶层冲突
- ❌ **第二季钩子**：短剧必须闭环，不能留尾巴

## 7. 输出格式（创意阶段）

当复杂反转模式开启时，`creative` 步骤的输出应同时满足主链路 `creative` 模板（题材 / 世界观 / 视觉基调 / 核心主题 / Optional Upgrade）**和**下面这份反转骨架要求：

- 三幕段落每段 ≥ 250 字、含具体动作 / 对白 / 场景
- Act 1/2/3 合计明确写出至少 5 层反转的揭晓时刻（不要在文中标"第几层"，但读者要能数出来）
- 主角描述必须含 5 要素
- 海外向时剧名必须是英文，国内向时剧名仍为中文但反转结构同样适用
