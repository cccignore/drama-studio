# Hollywood Bilingual Script Rules

## Goal

Use a creator-friendly bilingual format for overseas short drama:

- Chinese for scene headings, action lines, camera cues, music cues and emotional parentheticals.
- English only for spoken dialogue.
- Character names use **single English names only** — no bilingual / dual-name format.

## Localization rules (hard constraint)

This format is for overseas short drama targeting English-speaking audiences (North America / Europe / Oceania / Latin America). The bilingual format is about **workflow** (Chinese 场记 for review), not about character ethnicity:

- Characters must be **non-Asian / Western**. Forbidden: Chinese / Korean / Japanese / Vietnamese / Thai / Filipino names, faces or cultural markers.
- Forbidden name patterns:
  - Chinese names: `林夏`, `陈辰`, `王伟`
  - Pinyin / romanized Asian names: `Lin Xia`, `Chen Chen`, `Park Min-jun`, `Nguyen Van A`
  - Bilingual dual-name format: `林夏 / Lin Xia`, `陈辰 / Chen Morrison`
  - Asian surnames in any English line: Chen / Wang / Li / Zhao / Park / Kim / Nguyen / Tran / Suzuki
- Use **single Western names** (First + Last). Examples: `Mia Carter`, `Lucas Reed`, `Ethan Walker`, `Sophia Hayes`, `Damian Cross`.
- Visual descriptions must not mention "黑长直 / 杏眼 / 东方面孔 / 旗袍 / 汉服 / 和服 / 韩服" or other Asian-coded markers.
- Settings must be Western (NYC / LA / Chicago / London / Sydney / Toronto / São Paulo) — not "沪上 / 京城 / 华尔街相亲角 / 春节回家".

If upstream material (e.g. a 红果 source drama) uses Asian names or settings, **replace them with Western equivalents**. Do not preserve them out of "loyalty to source".

## Format

```text
## 场 1 · Café on 5th Avenue（曼哈顿咖啡馆 / 日）

△ （特写）Mia 推门而入，风衣被风吹得贴在身上，呼吸还没平。
♪ 低沉钢琴声压住两人的呼吸。
**Mia**（惊讶）: "Damian? What are you doing here?"
**Damian**（冷静）: "I've been waiting for you. For three years."
```

注意：

- 角色名一行只写一个英文主名 `**Mia**`，**不要**写 `**Mia / 米娅**` 或 `**林夏 / Lin Xia**`。
- 场记里出现角色名时也用同一个英文主名（如 `△ Mia 推门而入`），不要写中文名。

## Dialogue Rules

- English dialogue should be short, direct, emotional and platform-friendly.
- Avoid literal translation from Chinese idioms.
- Avoid mixed-language lines such as "I am very 生气" or "Don't 骗我".
- Do not use prose tags like "she said" or "he replied"; dialogue must be direct speech.

## Pacing

- Open with a visual hook in the first 5 seconds.
- Escalate conflict every scene.
- End on a payoff plus a new hook, or hard-cut at the paywall moment.
